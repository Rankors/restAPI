    Этот проект реализаует простой RESTAPI, для работы чатов
Стек:
    Golang как язык программирования
    net/http библиотека для работы по протоколу http
    GORM для работы с субд - postgreSQL - База данных
    testify для тестов 
Функционал:
    Создание чатов
    Отправка сообщений в определенный чат по id чата
    Получение N сообщений из чата по id чата (N не меньше 0 и не больше 100, стандарт 20 )
    Удаление чата, со всеми его сообщениями
